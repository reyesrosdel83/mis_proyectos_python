# Cálculo de interés bancario 💰

Programa en Python que calcula el capital total obtenido según una inversión inicial, la tasa de interés anual y el tiempo de inversión.

Es un ejercicio práctico para aprender el uso de funciones, entrada de datos y formato de salida en Python.

## Características

- Solicita inversión inicial, tasa de interés y tiempo.
- Convierte el porcentaje de interés a decimal.
- Calcula la ganancia anual.
- Calcula el capital total al final del plazo.
- Muestra un desglose final ordenado y formateado.

## 🛠️ Requisitos

- Python 3.8 o superior

No necesita librerías externas.
